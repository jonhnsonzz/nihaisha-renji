## 六、跨体系查询路径

### 场景1：已知八字→推体质

```
八字排盘 → 命宫主星 → 查"主星→体质倾向表" 
         → 疾厄宫星曜 → 查"杀星→疾病指向" 
         → 化忌宫位 → 查"宫位→脏腑" 
         → 进入人纪按对应脏腑辨证
```

### 场景2：已知疾病→推命理原因

```
疾病（如心脏病）→ 对应午宫（心脏） 
                → 查紫微斗数午宫星辰 
                → 看是否命盘有心脏病暗示 
                → 查阳宅是否西北角为厨房
```

### 场景3：综合调养

```
天机道（知先天体质倾向）
  + 人纪（辨证论治/用药）
  + 地脉道（调整阳宅方位）
= 完整的身心健康方案
```

## 七、使用建议

1. **优先使用单个体系**：大多数问题在天纪或人纪各自的体系内就能解决
2. **跨体系联动**：当单一体系无法解释时（如"为什么这个八字的人反复患某病"），通过本图谱寻找交叉关联
3. **人纪为基础**：中医诊断（人纪）是健康管理的核心，命理（天纪）提供的是辅助参考和预防建议
4. **"疾厄宫+化忌"是最直接的切入点**：看命盘时先看疾厄宫和化忌所在宫位，就能快速锁定易患疾病方向
